# phonetic-dictionary
Python scripts for building a phonetic dictionar of fr.wiktionary.org.

Since the phonetic dictionnary is built from the content of wiktionary.org, it inherits its licence.
The dictionnary has been generated for this project: http://easypronunciation.com/fr/news/cr%C3%A9ons-un-dictionnaire-phon%C3%A9tique-fran%C3%A7ais%21-recherche-des-volontaires/28
